🌊 C'est quoi un signal ?

Avant de plonger dans le cœur humain, comprenons les bases ! Un signal, c'est comme une vague. Joue avec les curseurs pour comprendre comment l'Amplitude (la hauteur) et la Fréquence (la vitesse des vagues) modifient le signal... et le son !

1. Le Bac à sable (Découverte)

Modifie les paramètres et écoute le résultat. Plus la fréquence est élevée, plus le son est aigu !

2. Le Défi de Synchronisation 🎯

L'ordinateur a généré un signal cible (en pointillés gris). Utilise tes curseurs pour superposer exactement ton signal (en vert) sur le sien !

✨ Signal Synchronisé ! Bien joué ! ✨

3. Le Défi de la Période ⏱️

La **Période (T)** est le temps (en secondes) que met la vague pour faire un motif complet. Regarde la grille : chaque grande ligne verticale vaut 0.5 seconde. Ajuste ton curseur pour atteindre la période demandée !

Objectif : Atteindre une période 2.0 s

✨ Période exacte !

4. Le Spectre : La recette secrète du signal

Un signal complexe (comme une voix) est un mélange de plusieurs ondes simples. Le spectre permet de séparer les graves (à gauche) des aigus (à droite) pour voir exactement de quoi le son est composé.
