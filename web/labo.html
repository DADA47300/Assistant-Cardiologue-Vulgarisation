🩺 Labo ECG — à toi de jouer, ingénieur !

Le cardiologue t'envoie l'enregistrement du cœur d'un patient… mais le signal est sale et illisible. Ta mission : choisir le bon nettoyage, régler ta vue, et repérer toi-même les battements. Attention : on peut se tromper — il faudra réfléchir !

Défi 1 — Répare le signal !

L'enregistrement est illisible. Le tracé tangue de haut en bas et est plein de parasites. Lance d'abord le capteur, puis utilise la console pour trouver le bon réglage des filtres.

Clique pour démarrer l'acquisition du signal en temps réel.

🎛 Console de nettoyage

〰️ Filtre Anti-Tangage 0
Supprime les ondulations lentes dues aux mouvements du patient.
⚡ Filtre Anti-Grésillement 0
Supprime les interférences électriques et le bruit du capteur.
Qualité du signal 0 %

Signal brut — ajuste les filtres pour le nettoyer.

Défi 2 — Trouve le bon zoom

L'enregistrement dure 24 secondes : tout est tassé. Déplace le curseur pour choisir la largeur affichée. Trop large, on ne distingue rien ; trop serré, on perd la vue d'ensemble.

Large Serré 24 s
Affiche tout l'enregistrement pour l'instant…

Défi 3 — Repère les battements toi-même

À toi de jouer comme un cardiologue : clique sur le graphe, sur chaque grand pic (onde R) que tu repères. Reclique sur un point pour l'enlever. Quand tu as terminé, vérifie si tu as trouvé les bons !
